## Requirements ⚙️
- [Yarn](https://yarnpkg.com/)
- [Node.js](https://nodejs.org/en/)
  
## Installation ⬇️
```bash

$ yarn

```

## Running the app 🏃

  

```bash

# development

$ yarn start

  

# watch mode

$ yarn start:dev

  

# production mode

$ yarn start:prod

```

  

## Test 🧪

  

```bash

# unit tests

$ yarn test

  

# e2e tests

$ yarn test:e2e

  

# test coverage

$ yarn test:cov

```
